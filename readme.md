# Create switching glass content with web ui and javascript
When clicking glass that has content it will take the content and empty the glass.
then you can click on empty glass to place taken content to fill the empty glass.
note: you can't place a taken content on glass that has content

appear horizontally when screen has width more than 768px max 1024px
